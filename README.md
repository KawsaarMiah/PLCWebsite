# Team Website & LaTeX Presentation

A two-part group project built with a team of 4 Computer Science students at NYIT.

## Part 1 — Team Website
A multi-page team profile website built using HTML and CSS.
- Used flexbox for responsive layout
- Each member's photo, name, major, and year displayed on the page
- Collaborated using Git and VS Code LiveShare for real-time editing

## Part 2 — LaTeX Presentation
A 10-slide academic presentation built using LaTeX and the Beamer package on the topic of "What is LaTeX?"
- Covered the history, development, and use cases of LaTeX
- Used Beamer themes and structured slide formatting
- Compiled into a professional PDF presentation
